Welcome to Hydna Pong

Hydna Pong is a multiplayer, peer-to-peer, version of the classic Pong Game. The game uses Hydna as network provider. Please take a look a the source code. The source code is well documented and will provide you with some guidelines in how to write your own game with the help of Hydna.

Click the button bellow to search for an opponent to play with.

Use arrow key up and down to move your paddle.

Happy gaming!
